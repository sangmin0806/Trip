## 📘프로젝트 개요

🏷 **프로젝트 이름**: **나침반**

🗓️ **프로젝트 기간**: 2024.04.15 ~ 2024.05.24

👥 **구성원**: 이상민, 최승호

 **소개**: 코로나 이후 늘어난 여행 수요와 함께 여행관련 서비스의 수요도 함께 증가하고 있습니다.

이에따라 여행계획, 추천, 후기등의 커뮤니티가 있는 플랫폼을 기획하게 되었습니다.

## 📌주요기능

- #### [국내 관광지 정보 조회]
  - 지역별 관광지 조회
  - 관광유형별(문화시설, 축제행사, 음식점, ...) 관광지 조회

- #### [회원 관리]
  - 로그인/회원가입
  - 마이페이지
  - 회원정보 수정
  - 회원탈퇴
   
- #### [여행 계획]
  - 여행지 검색
  - 여행 계획 세우기
  - 나의 여행계획 보기
  - 해당 관광지 네이버 지도에서 확인하기

- #### [AI 추천]
  - 여행 컨셉에 맞는 여행 코스 추천
    
 - #### [여행 후기]
   - 후기 등록, 수정, 삭제, 조회
   - 작성자/제목으로 게시물 검색

 - #### [실시간 대화방]
   - 소켓통신을 활용한 채팅방
   - 실시간으로 대화 가능
